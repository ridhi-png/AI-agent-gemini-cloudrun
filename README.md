# AI Agent using Gemini + Cloud Run

## 🚀 Overview
This project implements a simple AI agent using Google Gemini that summarizes text input.

## 🧠 Features
- Uses Gemini API for text summarization
- Exposes an HTTP endpoint
- Deployed on Google Cloud Run

{
  "text": "Your long paragraph here"
